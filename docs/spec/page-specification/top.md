```
┌─────────────────────────────────────────────────────────────┐
│                        📚 Bookmarker                        │
│                                                             │
│         あなたのブックマークをスマートに管理                │
│                                                             │

│       [ ログイン ]         [ 新規登録 ]                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 機能紹介セクション

| アイコン | タイトル | 説明 |
|---------|---------|------|
| 🔖 | ブックマーク管理 | URLとファイルパスをまとめて管理 |
| 🏷️ | タグ整理 | 柔軟なタグ付けで素早く検索 |
| ⭐ | お気に入り | 重要なブックマークをすぐにアクセス |
| 🔒 | 安全な管理 | あなただけのプライベート空間 |

## 状態管理

### ログイン済みユーザーの場合

- ログイン/新規登録ボタンの代わりに「ダッシュボードへ」ボタンを表示
- `/bookmarks` へリダイレクト

---

## レスポンシブ対応

| ブレークポイント | 対応 |
|-----------------|------|
| モバイル（< 640px） | シングルカラム、ボタン縦並び |
| タブレット（640px - 1024px） | 2カラムグリッド |
| デスクトップ（> 1024px） | 4カラムグリッド |

---

## アクセシビリティ

- ヒーロー見出しは `<h1>` を使用
- ボタンには適切な `aria-label` を設定
- キーボードナビゲーション対応
