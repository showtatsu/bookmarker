## デザインガイドライン

### カラーリング

- 背景カラーは「黒(#000000)」
- メインコンテンツをカードで囲み、カードの背景カラーは 「白(#FFFFFF)」基調
- メインコンテンツ内のテキストカラーは 「黒(#000000)」
- ボタンカラーは 決定ボタンを 「青(#0000FF)」、危険な操作を行うボタンを 「赤(#FF0000)」とし、いずれも白抜きテキストとする

### フッター

- すべてのページに共通のフッターを配置
- フッターには著作権情報、利用規約、プライバシーポリシーへのリンクを含む
- コンパクトなデザインとし、ページの最下部に配置、スクロールにまかせて固定はしない

### ヘッダー

- ログイン時: タブメニュー形式で「ブックマーク」「タグ」を表示
- タイトル（「Bookmarker」）はログイン時は非表示
- 選択中のタブは背景色（青）と下線で強調
- `whitespace-nowrap` で折り返しを防止
- コンパクトな高さ48px（h-12）

### アイコン

- **SVGベースのアイコンを使用**（絵文字/フォントアイコンは使用禁止）
- 環境間の差異を吸収し、一貫した表示を保証
- アイコンコンポーネントは `/components/Icons.tsx` に集約
- 主要アイコン：
  - 検索: `SearchIcon`
  - お気に入り: `StarIcon` (filled/outlined)
  - 編集: `EditIcon`
  - 削除: `TrashIcon`
  - コピー: `CopyIcon`
  - 追加: `PlusIcon`
  - パスタイプ: `GlobeIcon` (URL), `FolderIcon` (ファイル), `ServerIcon` (ネットワーク), `FileIcon` (その他)
