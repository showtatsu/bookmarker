# Bookmarker - マルチユーザー対応ブックマーク管理アプリ

Next.js を使用したマルチユーザー対応のブックマーク管理アプリケーションです。
URLとファイルパスの両方をブックマークとして管理でき、タグベースの検索・フィルタリング機能を提供します。

## 機能

### 👥 マルチユーザー対応

- ユーザー登録・ログイン機能
- JWT認証
- ユーザー別データ分離
- APIトークン発行

### 📚 ブックマーク管理

- URLとファイルパスの両方に対応
- タイトル、説明、タグの設定
- お気に入り機能
- 作成日・更新日の自動記録

### 🔍 検索・フィルタリング

- タイトル、説明、URL、ファイルパスでの全文検索
- タグベースのフィルタリング
- タイプ別フィルタ（URL、ファイル、お気に入り）
- リアルタイム検索

### 🎨 UI/UX

- レスポンシブデザイン
- TailwindCSSによるモダンなUI
- 直感的な操作性
- 全体的にコンパクトで多くの情報を一画面に表示しやすいリストビュー

## セキュリティ機能

- JWT による認証
- bcryptjs によるパスワードハッシュ化
- Helmet による基本的なセキュリティヘッダー設定
- CORS 設定
- 入力バリデーション

## 技術スタック

### バックエンド

- **Node.js** - RESTful API サーバー
- **TypeScript** - 型安全な開発
- **Prisma ORM** - データベース操作
- **SQLite3** - 軽量データベース
- **PostgreSQL** - 将来的なスケーラビリティ対応
- **JWT** - 認証
- **bcryptjs** - パスワードハッシュ化

### フロントエンド

- **Next.js** - UI フレームワーク
- **TailwindCSS** - スタイリング

## セットアップ

### 必要条件

- Node.js 24.x 以上
- npm

### インストール手順

1. リポジトリをクローンします。

```bash
git clone
cd bookmarker
```

2. 依存関係をインストールします。

```bash
npm install
```

3. 環境変数ファイルを作成します。

```bash
cp .env.example .env
```

4. データベースをセットアップします。

```bash
npx prisma migrate dev --name init
```

5. 開発サーバーを起動します。

```bash
npm run dev
```
